Funktionen

  • Bemaßen des Boardes (der Umrisse und der Bohrungen)
  • Erstellen einer Bohrertabelle
  • Exportieren diverser Boardansichten
    • Layout Top / Bottom / Layer 2 / Layer 3
    • Bestückung Top / Bottom
    • Spiegeln der Bottom Ansichten
    • Beschneiden aller Bilder
  • Automatische Abfrage des Preises und des Lagerstandes (wenn vom Distributor unterstützt)
    • Farnell / Newark (Farnell.com)
    • Digikey (Digikey.com)
    • Sparkfun (Sparkfun.com)
    • Unterstützung für weitere Distributoren auf Anfrage
  • Exportieren der Stückliste
    • Automatische Erkennung des Bauteiltyps (Kondensator, Widerstand, etc.)
    • Automatische Sortierung nach Bauteiltyp und Wert
    • Unterstützte Formate CSV, XLS, XLSX, DBASE3
  • AutoProjectDocumentation Unterstützung

How To

Das Export ULP ist dazu gedacht auf schnellstem Wege von einem bestehendem Schematic und einem bestehenden Board zu einer vollständigen Projektdokumentation zu gelangen, um sicherzustellen das dies möglich ist müssen einige Dinge im Vorfeld bzw. bei der Erstellung des Stromlaufplans berücksichtigt werden.

Attribute setzen

Das Export ULP braucht eine Quelle für die Bestellnummern und die Distributoren der Bauteile, dies wird über Attribute realisiert.
Um die Attribute für ein Bauteil zu verändern bzw. einzustellen muss mit einem Rechtsklick auf das Bauteil geklickt werden.




Anschließen muss der Punkt "Attribute" selektiert werden, was folgendes Fenster öffnen sollte.




Nun müssen über den Menüpunkt "Neu" folgende Attribute hinzugefügt werden.
DISTRIBUTORDer Verkäufer des Bauteils, z.B. farnell.com oder digikey.com
ORDER_NODie Bestellnummer des Bauteils (beim Verkäufer), z.B. 1414610
MANUFACTURER* Hersteller des Bauteils, z.B. KEMET
PART_NO* Bauteilname laut Hersteller, z.B. C0603C104K4RAC
REFERENCE_NO** Frei wählbare Referenznummer, z.B. für firmen-internes Bestellsystem
* Nur für eine der HTL Vorgaben entsprechenden Stückliste erforderlich
** Optional

Bohrersymbole festlegen

Damit eine richtige Bohrertabelle erstellt werden kann, müssen außerdem noch die Bohrsymbole den entsprechenden Bohrerdurchmessern zugeordnet werden.
Dies wird über den Menüpunkt "Optionen" -> "Einstellungen" erledigt




Nun können die Bohrersymbole unter dem "Bohrersymbole" Tab verwaltet werden.




Zunächst müssen alle Bohrdurchmesser gelöscht werden, hierzu muss zunächst ein Bohrdurchmesser ausgewählt werden und danach auf den "Löschen" Button geklickt werden bis alle Durchmesser entfernt wurden. Mit einem Klick auf "Erzeugen" werden nun alle im akutellen Layout verhandenen Bohrdurchmesser angelegt.

Nun sollte alles für die Ausführung des Export ULP vorbereitet sein.
